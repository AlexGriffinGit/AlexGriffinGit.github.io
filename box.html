Box the Platformer

Box Is a physics based platformer that was developed in the ODBAR Engine as another part of that final year university module. Box was programmed in C++ using many of the features that the ODBAR Engine provided.

The premise behind box was a horror science fiction scenario in which the protagonist wakes up as a sentient box in a deserted underwater lab and must escape. They do this by reversing gravity to traverse the labyrinthian abandoned lab. They must avoid malfunctioning electrical equipment that will shock the player on touch, as if they sustain 3 hits the box and player would die and have to restart. On completing the game the main menu title colour would change to gold a can be seen in the last image of the above slideshow.

Box utilised many of the features that the ODBAR Engine provided, such as

  • I took full use of the physics engine and used it to create a challenging and bizarre platformer.
  • The saving and loading of levels was used to create 3 different levels that would each lead into each other with minimal loading times.
  • Different tile types were used to create the start and end of the level as well as the electrical hazards peppering the levels.
  • The sound engine was used to play harsh electrical cracking when near the hazards to tip the player off that they were dangerous.
I'm not comfortable in sharing the code for this project as my programming has improved greatly since this project and I don't think it's a fair representation of my skillset. I do plan on making a game with a similar premise in the future, most likely in the Unity Game Engine which will of course be listed on this portfolio.
