This Portfolio

This portfolio website has undergone two different variations since I first started developing it and learning web development in my spare time. I believe I have improved greatly both in terms of HTML & CSS syntax as well as UI design.

The first variation of my portfolio was monochromatic and was written in vanilla HTML and CSS with a little bit of Javascript. I started developing it as a place to show off my projects both past and present, as well as wanting to learn more about web development in order to further expand my skillset. I was very happy with the website that I had created with HTML and CSS, especially as I was new to web development (and still am). I noted a number of things that needed improvement throughout the site and especially when looked at on a mobile device. If you haven't already there are some screenshots of the old portfolio available in the images above.

After some research I learned about Bootstrap and what it could do to make development simpler as well as address the responsiveness for different devices.

After creating a Bootstrap theme I started moving my existing portfolio pages over to the new Bootstrap style, which is the current iteration and what you're reading now. The site evolved over time and now is much more suitable for different devices and screen sizes. I found developing with Bootstrap very rewarding and I'm very happy with my portfolio in it's current state.

I've learnt a lot about web development through creating this portfolio page but i feel like I've only scratched the surface and there's a lot more I can do. Especially in regards to Javascript, I definitely aim to experiment and try different things out in the future.

Both versions of my portfolio are available to look through and download on my Github.
